\# Paid Parking System – React Frontend



A modern, responsive \*\*React frontend\*\* for the Paid Parking Management System.  

This client application provides an interactive parking map and full user flow for booking and managing parking spots.



\## Features



\- \*\*Interactive parking map\*\* with 200 spots divided into 5 blocks (A–E)

\- Visual status indicators:

&nbsp; - Green – available spots

&nbsp; - Red – occupied spots

&nbsp; - Gray – placeholder spots

\- \*\*User authentication\*\* – registration and login with JWT

\- \*\*One-click booking\*\* – select a free spot → confirmation modal → instant booking (fake payment)

\- \*\*Booking cancellation\*\* – dedicated modal with confirmation

\- \*\*Real-time map updates\*\* – the map refreshes after booking/cancellation

\- \*\*Responsive design\*\* – fully adapted for mobile, tablet, and desktop

\- \*\*Beautiful dark theme\*\* with gradients, shadows, hover effects



\## Tech Stack



\- \*\*Framework\*\*: React 18

\- \*\*Build tool\*\*: Vite (fast development with HMR)

\- \*\*Styling\*\*: Tailwind CSS

\- \*\*HTTP client\*\*: Axios

\- \*\*Backend\*\*: Connected to \[Paid Parking Backend](https://github.com/AntonMuzhytskyi/paid-parking-system)



\## How to Run



```bash

git clone https://github.com/AntonMuzhytskyi/paid-parking-frontend.git

cd paid-parking-frontend

npm install

npm run dev

```



App available at http://localhost:5173

Backend must be running at http://localhost:8080
